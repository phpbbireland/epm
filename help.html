üîó Linking to Local Folders

You can link to folders on your computer or network using file:// URLs.

Windows Paths:
  • Local Drive: file:///C:/Projects/Arduino
  • Network Share: file:////server/shared/projects
  • Another Drive: file:///D:/Documents/Electronics
Mac/Linux Paths:
  • User Folder: file:///home/username/projects
  • Documents: file:///Users/username/Documents/Arduino
  • Root Path: file:///var/data/electronics
Important Rules:
  • ‚úÖ Always use forward slashes /
  • ‚úÖ Windows paths need three slashes: file:///
  • ‚úÖ Network shares need four slashes: file:////
  • ‚ùå Don't use backslashes \
  • ‚ùå Don't forget the protocol file://
Converting Windows Paths:

  • Browser Extensions: Some extensions allow file:// links (check your browser's extension store)
Best Practices:
