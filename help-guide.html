Understanding the Category Hierarchy

The system supports two-level category organization:

  • Parent Categories: Top-level categories (e.g., "Microcontrollers", "Sensors")
  • Subcategories: Nested under parent categories (e.g., "Arduino" under "Microcontrollers")

Creating Categories

  1. Go to Categories page
  2. Click "â• Add New Category"
  3. Enter category name and description
  4. Leave "Parent Category" as "None" for top-level category
  5. OR select a parent to create a subcategory
  6. Click "Add Category"

Managing Subcategories

  1. On the Categories page, find your parent category
  2. Click the blue subcategories button (shows count like "3 subs")
  3. You'll be taken to the subcategories page for that parent
  4. Add, edit, or delete subcategories as needed
  5. Click "â† Back to Categories" to return

â“ Troubleshooting

Common Issues

Q: File links don't open when clicked

A: Modern browsers block file:// links for security. That's why folder badges copy the link instead! Just paste it into your file explorer or browser address bar.

Q: Can't delete a category

A: You can only delete categories that aren't in use. Move or delete parts in that category first, or delete its subcategories.

Q: Stock badges show wrong colors

A: Check your Low Stock Threshold in Settings. The default is 5 items.

Q: Changes don't appear

A: Try refreshing the page. All changes are saved to the database immediately.
